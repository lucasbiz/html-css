Parágrafos e quebras de linha

Você pode escrever um paragrafo de qualquer jeito.

Para criar um novo parágrafo tem que colocar a tag de novo

para criar uma quebra de linha, basta colocar a tag
"br", que significa "break roll"

Para colocar os simbolos que criam tags em html sem
que as tags em si sejam criadas, basta colocá-las entre os símbolos

vamos colocar alguns símbolos
® (marca registrada)
© (cópia)
™ (trade mark)
